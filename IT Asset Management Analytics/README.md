# Enterprise IT Infrastructure Operations Analytics Dashboard
Power BI | IT Asset Management | Network Operations | APAC Infrastructure

This project presents a Power BI dashboard designed to provide operational visibility into enterprise IT infrastructure assets across the APAC region.

The dashboard enables management to monitor infrastructure distribution, network ownership (WAN/LAN), security implementation, and infrastructure coverage across multiple countries and locations.

Key analytics include:
• Infrastructure footprint across countries and locations
• Network management ownership (WAN & LAN)
• Infrastructure type distribution by entity
• Asset distribution across operational regions
• Security infrastructure deployment insights

The dashboard helps IT infrastructure teams improve infrastructure governance, operational monitoring, and decision-making.

## Data Note
The dataset used in this dashboard originates from enterprise IT infrastructure systems and cannot be shared publicly due to confidentiality restrictions. The repository includes dashboard screenshots and analytical descriptions for demonstration purposes.
