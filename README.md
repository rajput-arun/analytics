# IT Operations Analytics Portfolio

**Power BI | SQL | Python | IT Service Management | IT Infrastructure Analytics**

This repository contains a collection of **data analytics and business intelligence dashboards focused on IT operations and service management environments**.

The projects demonstrate how operational data from enterprise IT systems can be transformed into **actionable insights for management decision-making** using modern analytics tools such as **Power BI, SQL, Python, and Excel**.

The dashboards focus on operational analytics areas such as:

* IT Service Management (ITSM)
* Service Desk Performance
* IT Infrastructure Visibility
* IT Asset Management
* Network Operations Monitoring

# Projects Included

## 1. Enterprise IT Infrastructure Operations Analytics Dashboard

**Tool:** Power BI
**Domain:** IT Asset Management / Infrastructure Operations

This dashboard provides operational visibility into **enterprise IT infrastructure assets across multiple countries and locations within the APAC region**.

### Key Insights Provided

* Infrastructure footprint across countries and locations
* Network ownership analysis (WAN / LAN management)
* Infrastructure distribution by entity code
* Security infrastructure implementation tracking
* Operational coverage across global locations

### Business Value

The dashboard helps IT leadership teams:

* monitor infrastructure distribution
* track network ownership responsibilities
* improve operational governance
* support infrastructure planning and capacity decisions

### Screenshots

Dashboard screenshots are available in the project folder.


## 2. IT Service Management Analytics Dashboard

**Tool:** Power BI
**Domain:** Service Desk Operations

This dashboard analyzes **service desk ticket data to identify operational trends, service performance, and support workload distribution.**

### Key Analytics

* Monthly ticket volume trends
* Priority-wise incident distribution
* Category-wise ticket analysis
* Agent performance monitoring
* SLA performance tracking

### Business Value

The dashboard enables IT operations teams to:

* identify recurring incidents
* monitor service performance
* optimize service desk workload
* improve incident response and resolution efficiency
